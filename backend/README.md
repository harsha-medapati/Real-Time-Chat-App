# Real-Time Chat App - Backend

This is the backend server for the Real-Time Chat Application built with Node.js, Express, and Socket.io.

## Getting Started

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

## Project Structure

- `config/` - Configuration files for database and Socket.io
- `models/` - MongoDB models for User, Message, and Group
- `routes/` - API endpoints
- `controllers/` - Business logic
- `middlewares/` - Custom middleware functions
- `utils/` - Utility functions
